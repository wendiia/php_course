В PHP имеется логический (boolean) тип, имеющий два значения - true (истина) и false (ложь).
Изучите с помощью var_dump() следующие выражения и объясните их.

1 == 1.0 - true, т.к. без сравнения типа значения
1 === 1.0 - false, т.к. тип int != float
'02' == 2 - true, т.к. php при сравнении приводит '02' в 2, но типы не сравниваются
'02' === 2 - false, string != int
'02' == '2' - true, т.к. php приводит обе строки к числу 2
